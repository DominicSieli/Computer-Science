Time Complexity Analysis
- Access Complexity: Constant Time O(N)
- Searching Complexity: Linear Time O(N)
- Inserting Complexity: Linear Time O(1)
- Deleting Complexity: Linear Time O(1)

Functions
- Push(T) O(1)
- T Pop() O(1)
- T Peek() O(1)
- bool Contains(T) O(N)

Queue:
Common operations:
Enqueue: Adds a new item to the back of the queue.
Dequeue: Removes the first item from the front of the queue.
IsEmpty: Checks if the queue is empty.
